# Employee Personal Account Monitoring & Security Control System

## Overview
This system monitors employee activity, detects personal account usage, and generates alerts and reports for administrators.

## Features
- Activity logging
- Alert detection (Low / Medium / High risk)
- Detection rule management
- Report generation and filtering
- Add/update detection rules

## Technologies
- Python (Tkinter GUI)
- SQLite database

## How to Run
```bash
python3 monitoring_system.py
